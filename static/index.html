How it works

Our platform decouples data extraction, search indexing, and multi-agent synthesis into a high-throughput pipeline.

  • 1. Ingestion Pipeline: Apache Airflow schedules metadata extractions from arXiv. PDFs are parsed via thread-isolated Docling parsers and loaded into PostgreSQL.
  • 2. Hybrid Search Indexing: Chunks are vectorized using Jina AI embeddings and stored in OpenSearch alongside BM25 mappings. Searches run fused Reciprocal Rank Fusion (RRF) scores.
  • 3. Multi-Agent Reasoning: A LangGraph state-graph validates inputs, retrieves context, grades document relevance, refines queries via LLM rewrites, and handles out-of-scope requests.
